[Akitect-comfyauto 사용자 가이드]

안녕하세요! Akitect-comfyauto를 이용해 주셔서 감사합니다.
본 가이드는 프로그램 설치부터 고품질 영상 스토리보드 및 이미지 생성까지의 과정을 누구나 쉽게 따라 할 수 있도록 정리한 안내서입니다.

1. 프로그램 소개
   - AI를 활용하여 시나리오 초안 설계, 세부 컷 구성, 그리고 고품질 이미지를 자동으로 생성해 주는 종합 콘텐츠 제작 도구입니다.

2. 시작 전 준비 사항 (최초 1회)
   - 본 프로그램은 실행 시 필요한 환경(Python, Node.js)을 자동으로 감지하고 설치를 시도합니다.
   - 하지만 원활한 실행을 위해 인터넷 연결 상태를 확인해 주세요.

3. 실행 방법
   - 프로젝트 폴더 내에 있는 'run_demo.ps1' 파일을 마우스 오른쪽 버튼으로 클릭합니다.
   - 'PowerShell에서 실행'을 선택합니다.
   - 검은색 콘솔 창이 뜨며 필요한 설정이 자동으로 진행됩니다.
   - 설정이 완료되면 자동으로 브라우저가 열리거나, 브라우저 주소창에 아래 주소를 입력하여 접속할 수 있습니다.
     * 사용자 화면: http://localhost:3500

4. 주요 환경 설정 (최초 실행 시)
   화면 좌측 하단 또는 설정(Settings) 메뉴에서 다음 정보를 입력해야 합니다.
   - OpenAI API Key: 스토리를 생성하는 AI의 두뇌 역할을 합니다. (구매하신 키를 입력하세요)
   - ComfyUI 경로: 이미지를 생성하는 프로그램의 설치 경로를 지정합니다.
     (예: C:\ComfyUI_windows_portable\ComfyUI)

5. 사용 단계 (워크플로우)
   [1단계: 스토리 초안 생성]
   - 원하는 카테고리를 선택하거나 자유롭게 주제를 입력하고 '생성'을 누릅니다.
   - AI가 10가지의 다양한 스토리 초안을 제안합니다.

   [2단계: 스토리 확정]
   - 마음에 드는 초안을 선택하면 AI가 이를 바탕으로 상세한 '컷(Cut)' 정보로 확장합니다.
   - 각 컷의 설명, 감정 수치, 카메라 앵글 등을 확인할 수 있습니다.

   [3단계: 이미지 생성]
   - 확정된 스토리를 바탕으로 '이미지 생성' 버튼을 누릅니다.
   - 설정한 ComfyUI를 통해 고퀄리티 실사 이미지가 차례대로 생성됩니다.

6. 팁 및 참고 사항
   - 이미지 생성 중 ComfyUI가 백그라운드에서 실행 중이어야 합니다.
   - 생성이 완료된 이미지는 'backend/outputs' 폴더에서 확인할 수 있습니다.
   - 문제가 발생할 경우 콘솔 창의 오류 메시지를 확인하시거나 관리자에게 문의해 주세요.

즐거운 창작 활동 되시길 바랍니다!
